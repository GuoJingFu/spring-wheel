#spring-wheel 打造属于自己的开发脚手架

##简介
　　工作快满一年了，此项目是对自己工作与学习的小结，计划持续维护下去，希望和大家多多交流。
　　该项目采用springboot框架，集成mybatis和shiro，采用velocity作为前端模板，采用maven 管理依赖，采用xml的配置方式。
##技术选型
|名称|版本|
|----|----|
|spring boot|1.2.7.RELEASE|
|spring mvc|4.2.0.RC2|
|shiro|1.2.3|
|mybats|3.2.8|
|通用maper|3.3.0|
|mybatis分页插件|4.0.0|
